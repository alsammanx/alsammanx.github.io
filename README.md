# Alaa Al Samman Portfolio

Static one-page portfolio prepared for GitHub Pages.

## Publish

1. Create a public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open Settings → Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save.
6. Under Custom domain, enter: alsammanx.com
7. Configure the DNS records at your domain provider.
8. After DNS verification, enable Enforce HTTPS.

The root of the repository must contain:
- index.html
- CNAME
- assets/
